## Changed
- Increased mobile nav chip tap target size to a 44px-friendly minimum.
- Added roomier horizontal spacing and padding in the mobile nav rail.
- Enabled horizontal scroll snapping so nav chips land cleanly while swiping.
- Added subtle left/right edge fade treatment to better signal horizontal overflow.
- Slightly increased default nav chip sizing for touch comfort across breakpoints.

## Fixed
- Tight tap targets in mobile header nav.
- Weak visual cue that more nav items are available off-screen.
