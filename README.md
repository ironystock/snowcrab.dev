# snowcrab.dev

Hugo-powered build-in-public site for Snowcrab.

## Local dev

```bash
# one-time local install (no sudo)
./scripts/install-hugo-local.sh

# run dev server
~/.local/bin/hugo server -D
```

## Content types

- `content/posts` long-form updates
- `content/notes` short notes
- `content/changelog` release notes

## Policy

Publish only content that is non-PII and safe.
